# 🦁 哺乳纲演化导航网站

一个交互式的哺乳动物演化探索网站，通过决策树方式探索 167 个科的分类单元。

## ✨ 功能特性

### 🌳 决策树导航
- 基于特征描述的选项，而不是直接显示名称
- 支持回溯和重新开始探索
- 面包屑导航显示完整路径

### 📖 图鉴系统
- 记录用户解锁的所有节点
- 显示解锁进度和百分比
- 点击已解锁节点可跳转到对应位置

### 🎉 解锁提示
- 解锁新物种时弹出精美提示
- 显示物种图标、名称和描述
- 3秒自动关闭或手动关闭

### 💾 存档管理
- 自动保存到浏览器 localStorage
- 支持导出 JSON 格式存档
- 支持导入存档恢复进度

## 📊 数据规模

- **总计节点**：200+
- **叶子节点**：167 个科
- **覆盖范围**：
  - 原兽亚纲（鸭嘴兽、针鼹）
  - 有袋类（袋鼠、考拉、袋熊等）
  - 非洲兽总目（大象、海牛等）
  - 灵长总目（猴、猿等）
  - 劳亚兽总目（蝙蝠、猫、狗、牛等）

## 🚀 快速开始

### 直接使用
直接用浏览器打开 `index.html` 即可使用！

### 本地服务器（推荐）
```bash
# Python 3
python3 -m http.server 8080

# 或者使用 Node
npx serve .
```
然后访问 http://localhost:8080

## 🎨 技术特点

- **纯前端实现**：无需后端服务器
- **响应式设计**：适配桌面端和移动端
- **本地存储**：localStorage 保存进度
- **模块化开发**：数据、渲染、逻辑分离
- **动画效果**：流畅的过渡和交互体验

## 📝 使用说明

1. 点击选项探索演化树
2. 使用面包屑导航回溯
3. 点击右上角图鉴查看已解锁物种
4. 点击图鉴中的已解锁节点可跳转
5. 使用存档功能保存/恢复进度
